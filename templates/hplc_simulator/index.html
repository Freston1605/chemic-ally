Learn chromatography by tuning parameters and seeing results.

How to Play
1. Choose Parameters

Select mobile phase composition, column type, and operational settings to separate the analyte mixture.

2. Run the Simulation

Click "Inject" to run the simulation. Watch the chromatogram develop as peaks elute from the column.

3. Optimize Your Score

Achieve baseline resolution (Rs >= 1.5) for all peaks in the shortest run time. Watch for overpressure!
